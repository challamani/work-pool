# Work Pool UI

## Purpose
React frontend for authentication, task marketplace, bidding, messaging, payments, and ratings flows.

## Local run
```bash
cd /home/runner/work/work-pool/work-pool/work-pool-ui
npm install
npm run dev
```

## Build and lint
```bash
npm run lint
npm run build
```

## Important env
- `VITE_API_BASE_URL` (default: `http://localhost:8080`)
- `VITE_WS_URL` (default: `http://localhost:8083`)
